# Template README Guide Book Game

Dokumen ini adalah template struktur **guide book** untuk game. Struktur ini bisa digunakan sebagai acuan saat membuat buku panduan penggunaan game, mulai dari pengenalan game, tampilan awal, fitur utama, cara bermain, sampai informasi pengembang.

---

## 1. Cover

Bagian cover berisi identitas utama guide book.

**Isi yang disarankan:**

- Judul: `Buku Panduan [Nama Game]`
- Nama game
- Nama penyusun/pengembang
- Logo game
- Karakter utama atau visual utama game

**Contoh:**

```text
BUKU PANDUAN
[Nama Game]

Disusun oleh:
[Nama Pembuat]
```

---

## 2. Tentang Game

Bagian ini menjelaskan gambaran umum game.

**Isi yang disarankan:**

- Nama game
- Tema game
- Tujuan game
- Target pemain
- Ringkasan fitur utama
- Manfaat game bagi pemain

**Contoh:**

```text
[Nama Game] adalah game pembelajaran interaktif yang bertujuan untuk membantu pemain memahami [topik utama game]. Game ini menyajikan materi dalam bentuk cerita, tantangan, kuis, dan permainan interaktif agar proses belajar menjadi lebih menyenangkan.
```

---

## 3. Tampilan Awal Game

Bagian ini menjelaskan tampilan pertama saat game dibuka.

**Isi yang disarankan:**

- Splash screen/loading screen
- Tampilan home/menu utama
- Tombol utama yang tersedia
- Tombol sound/mute
- Tombol keluar atau pengaturan

**Contoh struktur penjelasan:**

```text
Ketika pemain membuka game, akan muncul tampilan splash screen. Setelah itu, pemain diarahkan ke tampilan home yang berisi beberapa tombol utama seperti Mulai, Petunjuk, Pengaturan, dan Keluar.
```

---

## 4. Petunjuk Tombol

Bagian ini menjelaskan fungsi setiap tombol atau ikon yang ada dalam game.

| Tombol/Icon | Fungsi |
|---|---|
| Home | Kembali ke menu utama |
| Back | Kembali ke halaman sebelumnya |
| Next | Lanjut ke halaman berikutnya |
| Sound On | Menyalakan suara game |
| Sound Off | Mematikan suara game |
| Start | Memulai permainan |
| Exit | Keluar dari fitur/game |
| Retry | Mengulang permainan |
| Reset | Menghapus progress/skor sementara |

---

## 5. Menu Utama

Bagian ini menjelaskan isi dari menu utama game.

**Isi yang disarankan:**

- Tombol Story/Cerita
- Tombol Level/Misi
- Tombol Quiz
- Tombol Game/Challenge
- Tombol Profil
- Tombol Sumber Asset
- Tombol Pengaturan

**Contoh:**

```text
Pada menu utama, pemain dapat memilih beberapa fitur utama, seperti Cerita, Quiz, Game, Petunjuk, dan Profil Pengembang. Setiap tombol akan mengarahkan pemain ke halaman yang berbeda sesuai fungsinya.
```

---

## 6. Fitur Cerita / Story Mode

Bagian ini menjelaskan alur fitur cerita dalam game.

**Isi yang disarankan:**

- Cara masuk ke fitur cerita
- Urutan cerita
- Dialog karakter
- Pilihan misi atau musuh
- Cara lanjut ke cerita berikutnya
- Kondisi cerita selesai

**Contoh:**

```text
Pemain dapat masuk ke fitur cerita dengan menekan tombol Cerita pada menu utama. Pada fitur ini, pemain akan membaca dialog antar karakter dan mengikuti alur cerita sampai misi selesai.
```

---

## 7. Fitur Level / Misi / Battle

Bagian ini menjelaskan tantangan utama dalam game.

**Isi yang disarankan:**

- Cara memilih level/misi
- Jenis tantangan
- Cara menyelesaikan tantangan
- Kondisi menang
- Kondisi kalah
- Tombol untuk mengulang atau kembali ke menu

**Contoh:**

```text
Pada fitur battle, pemain akan diberikan pertanyaan atau tantangan. Pemain harus memilih jawaban atau aksi yang tepat untuk mengalahkan musuh. Jika jawaban benar, pemain dapat lanjut ke tahap berikutnya. Jika salah, pemain dapat mencoba kembali.
```

---

## 8. Fitur Quiz

Bagian ini menjelaskan sistem kuis dalam game.

**Isi yang disarankan:**

- Pilihan level kuis
- Jumlah soal
- Sistem skor
- Sistem nyawa
- Durasi waktu
- Indikator jawaban benar/salah
- Tombol retry

**Contoh:**

```text
Fitur Quiz terdiri dari beberapa level, seperti Easy, Medium, dan Hard. Setiap level berisi beberapa pertanyaan. Pemain akan mendapatkan skor jika menjawab dengan benar dan kehilangan nyawa jika menjawab salah.
```

---

## 9. Fitur Game / Mini Game

Bagian ini menjelaskan gameplay utama atau mini game.

**Isi yang disarankan:**

- Cara memulai game
- Cara menjawab atau memainkan game
- Aturan permainan
- Sistem skor
- Kondisi benar/salah
- Tombol lanjut, ulangi, menu, atau restart

**Contoh:**

```text
Pemain dapat menekan tombol Start untuk memulai mini game. Pemain harus menjawab pertanyaan dengan memilih huruf, item, atau opsi yang tersedia. Jika jawaban benar, pemain dapat lanjut ke pertanyaan berikutnya. Jika salah, pemain dapat mengulangi jawaban.
```

---

## 10. Item, Bantuan, atau Power-Up

Bagian ini menjelaskan bantuan yang dapat digunakan pemain.

| Bantuan/Item | Fungsi | Biaya/Syarat |
|---|---|---|
| Bantuan 1 | Menghilangkan pilihan yang salah | 5 koin |
| Bantuan 2 | Menampilkan satu petunjuk jawaban | 5 koin |
| Bantuan 3 | Menampilkan jawaban | 10 koin |

**Contoh penjelasan:**

```text
Pemain dapat menggunakan bantuan ketika kesulitan menjawab soal. Setiap bantuan memiliki fungsi berbeda dan membutuhkan jumlah koin tertentu untuk digunakan.
```

---

## 11. Sistem Skor, Koin, dan Progress

Bagian ini menjelaskan mekanisme penilaian dalam game.

**Isi yang disarankan:**

- Cara mendapatkan skor
- Cara mendapatkan koin
- Fungsi koin
- Penyimpanan progress
- Jumlah skor maksimal
- Kondisi progress bertambah atau dihapus

**Contoh:**

```text
Skor akan bertambah setiap kali pemain menjawab pertanyaan dengan benar. Koin dapat digunakan untuk membeli bantuan selama permainan berlangsung.
```

---

## 12. Kondisi Menang dan Kalah

Bagian ini menjelaskan akhir dari permainan.

| Kondisi | Penjelasan |
|---|---|
| Menang | Pemain berhasil menyelesaikan semua misi/soal |
| Kalah | Pemain kehabisan nyawa, waktu habis, atau gagal menyelesaikan tantangan |
| Game Complete | Semua level atau cerita telah selesai |
| Game Over | Pemain gagal dan harus mengulang permainan |

---

## 13. Reset, Restart, dan Exit

Bagian ini menjelaskan cara mengatur ulang atau keluar dari game.

**Isi yang disarankan:**

- Tombol Reset Game
- Tombol Restart
- Tombol Exit
- Apa yang terjadi setelah reset
- Cara kembali ke menu utama

**Contoh:**

```text
Jika pemain ingin menghapus skor, jawaban, atau progress sebelumnya, pemain dapat menekan tombol Reset Game. Jika ingin keluar dari fitur game dan kembali ke menu utama, pemain dapat menekan tombol Exit.
```

---

## 14. Profil Pengembang

Bagian ini berisi informasi pembuat game.

**Isi yang disarankan:**

- Nama pengembang
- Kelas/jurusan/institusi
- Email atau kontak
- Peran dalam pengembangan
- Foto atau avatar pengembang jika diperlukan

**Contoh:**

```text
Nama       : [Nama Pengembang]
Peran      : Game Developer
Institusi  : [Nama Sekolah/Kampus]
Kontak     : [Email/Kontak]
```

---

## 15. Informasi Sumber Asset

Bagian ini mencantumkan sumber media dan tools yang digunakan.

**Isi yang disarankan:**

- Sumber gambar
- Sumber ikon
- Sumber musik/sound effect
- Sumber video
- Font yang digunakan
- Game engine atau software pengembangan

**Contoh:**

```text
Game Engine : Unity
Desain UI   : Canva/Figma
Asset Gambar: [Sumber Asset]
Musik/SFX   : [Sumber Audio]
Font        : [Nama Font]
```

---

## 16. Format Penyusunan Guide Book

Agar guide book mudah dipahami, setiap halaman sebaiknya memiliki pola berikut:

```text
Judul halaman
Screenshot tampilan game
Penjelasan singkat fungsi halaman
Langkah penggunaan
Keterangan tombol
Catatan tambahan jika diperlukan
```

---

Berikut urutan halaman yang bisa digunakan:

1. Cover
2. Tentang Game
3. Tampilan Splash Screen
4. Tampilan Home/Menu Utama
5. Petunjuk Tombol
6. Informasi Game
7. Profil Pengembang
8. Informasi Sumber Asset
9. Fitur Cerita/Story Mode
10. Fitur Level/Misi/Battle
11. Fitur Quiz
12. Fitur Mini Game
13. Bantuan/Power-Up
14. Sistem Skor dan Koin
15. Reset/Restart/Exit
16. Penutup

---

## 18. Checklist Guide Book

Gunakan checklist ini sebelum guide book dikumpulkan atau dipublikasikan.

- [ ] Cover sudah memuat judul, nama game, dan nama pembuat
- [ ] Deskripsi game sudah jelas
- [ ] Semua menu utama dijelaskan
- [ ] Semua tombol dijelaskan fungsinya
- [ ] Setiap fitur memiliki screenshot
- [ ] Cara bermain dijelaskan secara berurutan
- [ ] Kondisi menang dan kalah dijelaskan
- [ ] Sistem skor/koin/progress dijelaskan
- [ ] Reset, restart, dan exit dijelaskan
- [ ] Profil pengembang dicantumkan
- [ ] Sumber asset dicantumkan
- [ ] Bahasa mudah dipahami oleh pemain baru

---

## 19. Catatan

Template ini dapat disesuaikan dengan genre game masing-masing. Jika game memiliki fitur tambahan seperti inventory, leaderboard, shop, achievement, multiplayer, atau save/load, tambahkan bagian khusus untuk menjelaskan fitur tersebut.
